🚀 PRÓXIMAS AÇÕES (Execute Agora)
═══════════════════════════════════════════════════════════════════════════════

1. Abra o terminal na pasta: C:\Users\liane\Documents\M\Avoar Site\sistema

2. Execute os comandos abaixo (um por um):

   a) Criar banco de dados:
      psql -U postgres -c "CREATE DATABASE avoar_db;"

   b) Criar tabelas (Windows):
      psql -U postgres -d avoar_db -f lib/db/schema.sql

      criar tabelas (macOS/Linux):
      psql -U postgres -d avoar_db < lib/db/schema.sql

   c) Testar conexão:
      npx ts-node lib/db/test-connection.ts

   d) (Opcional) Criar dados de exemplo:
      node seed-db.js

   e) Rodar o projeto:
      npm run dev

3. Acesse: http://localhost:3000

📞 TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

Problema:            Solução:
─────────────────────────────────────────────────────────────────────────────
PostgreSQL not found → Instale em: https://www.postgresql.org/download/
Porta 3000 em uso    → npm run dev -- -p 3001
"database not exist" → Crie com: CREATE DATABASE avoar_db;
"connection refused" → PostgreSQL não está rodando
"password auth fail" → Verifique .env.local

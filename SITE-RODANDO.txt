🐛 SE ALGO NÃO FUNCIONAR
═══════════════════════════════════════════════════════════════════════════════

Problema: Página de login sem CSS
Solução:
   • Verificar se Next.js está rodando na porta 3001
   • Reiniciar servidor estático
   • Verificar console do navegador para erros

Problema: Porta 3000/3001 em uso
Solução:
   • Fechar aplicativos que usam as portas
   • Usar comando: taskkill /F /IM node.exe

Problema: Banco de dados não conecta
Solução:
   • Verificar se PostgreSQL está rodando
   • Verificar credenciais em sistema/.env.local
   • Executar: npx ts-node lib/db/test-connection.ts

Problema: Links quebrados no site
Solução:
   • Verificar paths relativos nos HTML
   • Limpar cache do navegador (Ctrl+Shift+Del)
   • Reiniciar servidor estático

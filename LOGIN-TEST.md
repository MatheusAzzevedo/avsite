## Como Usar

1. Acesse a URL do login
2. Preencha os campos com as credenciais acima
3. Clique em "Entrar"
4. Você será redirecionado para o dashboard administrativo

## Funcionalidades Disponíveis

- **Dashboard:** Visão geral do sistema
- **Excursões:** Criar, editar e gerenciar excursões
- **Blog:** Criar e gerenciar posts
- **Configurações:** Ajustes do sistema
- **Usuários:** Gerenciar usuários admin

## Primeiro Acesso

Na primeira vez que acessar:
- O seed do banco de dados será executado automaticamente
- Um usuário admin será criado com as credenciais acima
- Dados de exemplo serão inseridos no banco
- Você poderá começar a usar o sistema

## Suporte

Caso tenha problemas:
- Verifique se a API está online (veja o status em API: Online/Offline)
- Limpe o cache do navegador
- Tente em uma aba anônima
- Verifique a senha (ela é sensível a maiúsculas/minúsculas)

🔧 COMO O CSS É CARREGADO
═══════════════════════════════════════════════════════════════════════════════

1. Next.js carrega globals.css automaticamente
2. Cada rota carrega seu CSS específico:
   • /admin/* carrega admin.css via admin/layout.tsx
   • /admin/login carrega login.css via login/page.tsx
3. Tailwind CSS processa as diretivas
4. Fonts são carregadas do Google Fonts
5. Servidor estático faz proxy sem interferir no CSS
